# Projet-IncH.Class

Ce dépôt regroupe les exercices JavaScript réalisés dans le cadre de ma formation en développement web chez **Inch.Class**.

## 📁 Structure du projet

```
Projet-IncH.Class/
├── exercice.html    # Page HTML servant de support d'exécution
├── Exercice.js      # Ensemble des exercices JavaScript
├── .hintrc          # Configuration du linter (webhint)
└── README.md
```

## 🎯 Exercices couverts

Le fichier `Exercice.js` regroupe plusieurs petits exercices d'algorithmique en JavaScript, dont :

- **Afficher l'âge** : demande l'âge à l'utilisateur et indique s'il est mineur ou majeur.
- **Pair ou impair** : demande un nombre et indique s'il est pair ou impair.
- **Mois en toutes lettres** : convertit un chiffre (1 à 12) en nom du mois correspondant.
- **Nombre secret** : fait deviner un nombre aléatoire compris entre 0 et 100 à l'utilisateur.
- **Suite de Fibonacci** : affiche la suite de Fibonacci jusqu'à un rang donné.
- **Somme de nombres** : calcule la somme des nombres compris entre 0 et un nombre saisi.

> ⚠️ Certains exercices sont mis en commentaire dans le fichier et peuvent être activés un par un pour tester chaque fonction indépendamment.

## 🚀 Utilisation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/Tchokouaga25/Projet-IncH.Class.git
   ```
2. Ouvrir le fichier `exercice.html` dans un navigateur.
3. Cliquer sur le bouton **Saisir** pour lancer l'exercice actif dans `Exercice.js`.
4. Pour tester un autre exercice, décommenter la fonction souhaitée dans `Exercice.js` puis recharger la page.
